# HARD_CLAUDE.md

이 프로젝트에서 사용한 하드웨어 부품 목록과 배선 정보.

---

## 구매 부품 목록

### 1. Arduino Uno R3
- 메인 마이크로컨트롤러 보드
- BLE 모듈 및 L298N 모터 드라이버와 통신
- 업로드 시 HM-10의 D2/D3 선 잠깐 분리 필요

### 2. HM-10 BLE 모듈
- BLE 4.0 무선 통신 모듈
- UART 통신 방식 (9600bps)
- **전원 주의: 반드시 3.3V 연결 (5V 연결 시 파손)**
- 서비스 UUID: `ffe0`, 특성 UUID: `ffe1`

### 3. L298N 듀얼 H-브리지 모터 드라이버
- 좌/우 모터 2채널 독립 PWM 속도 제어
- **ENA, ENB 점퍼 제거 필수** (제거해야 PWM 속도 제어 가능)
- 12V 입력 단자에 배터리 연결, 5VOUT → Arduino 5V 전원 공급

### 4. 4WD RC카 섀시 키트
- DC 모터 4개 포함 (좌 2개, 우 2개)
- 좌측 모터 2개는 L298N 채널 A에 병렬 연결
- 우측 모터 2개는 L298N 채널 B에 병렬 연결

### 5. 18650 배터리 × 2
- 직렬 연결 → 7.4V (L298N의 12V 입력 단자에 연결)
- L298N 내부 강압으로 Arduino 5V 전원 생성

## 핀 배선표

| Arduino 핀 | 연결 대상 | 설명 |
|-----------|---------|------|
| 3.3V | HM-10 VCC | ⚠️ 반드시 3.3V |
| GND | HM-10 GND | 공통 GND |
| D2 | HM-10 TXD | BLE → Arduino 수신 |
| D3 | HM-10 RXD | Arduino → BLE 송신 |
| D5 (PWM) | L298N ENA | 좌측 모터 속도 |
| D6 (PWM) | L298N ENB | 우측 모터 속도 |
| D7 | L298N IN1 | 좌측 모터 방향 A |
| D8 | L298N IN2 | 좌측 모터 방향 B |
| D9 | L298N IN3 | 우측 모터 방향 A |
| D10 | L298N IN4 | 우측 모터 방향 B |
| 5V | L298N 5VOUT | L298N → Arduino 전원 수전 |
| GND | L298N GND | 공통 GND |

**배터리 연결:** 배터리(+) → L298N 12V 단자, 배터리(-) → L298N GND

---

## 주의 사항

- HM-10은 반드시 **3.3V** 전원 공급 (5V 연결 시 모듈 파손)
- L298N ENA/ENB 점퍼를 **제거**해야 PWM 속도 제어 가능
- Arduino에 코드 업로드 시 HM-10의 D2/D3 연결선을 잠깐 분리 후 업로드
- 18650 배터리 2개 직렬 연결 = 7.4V → L298N 12V 단자에 연결
